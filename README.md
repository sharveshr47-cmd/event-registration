# Event Registration System

The Event Registration System is a responsive and interactive frontend mini project developed using HTML5, CSS3, and Vanilla JavaScript. This application is designed to simplify the process of registering participants for various events through an organized and user-friendly digital platform.

The system provides an attractive event listing interface where users can view available events, explore event details, and complete the registration process by submitting personal information such as name, email, phone number, and selected event category. JavaScript-based form validation ensures that all required fields are properly entered before successful submission.

After registration, the application generates a confirmation screen displaying the participant details, registered event information, and a success notification to simulate a real-time booking experience. The project demonstrates practical implementation of modern event handling systems used in seminars, workshops, college fests, webinars, and public programs.

This mini project highlights essential frontend development concepts such as dynamic DOM manipulation, event-driven form handling, user input validation, responsive card design, registration summary generation, and smooth page transitions. With its professional interface and real-world workflow simulation, the Event Registration System serves as an ideal academic project for showcasing web application development skills.

## Key Features
- Dynamic event listing section
- User-friendly registration form
- Name, email, mobile, and event validation
- Seat/category selection
- Registration success confirmation page
- Participant summary display
- Responsive modern interface
- Smooth animations and transitions

## Technologies Used
- HTML5
- CSS3
- Vanilla JavaScript

## Purpose of the Project
This project was developed to simulate an online event enrollment platform where users can quickly register for available events and organizers can maintain a simplified digital registration workflow.

## Project Workflow
1. User views available events.
2. User selects an event to register.
3. User fills registration form with details.
4. Form validation checks required information.
5. Registration is submitted successfully.
6. Confirmation summary is displayed.

## How to Run Locally
1. Download or clone the repository.
2. Open the project folder in VS Code.
3. Run `index.html` using Live Server extension.
4. Access the Event Registration webpage in browser.
